2. DevSecOps Automated Deployment Pipeline

Built a fully automated DevSecOps CI/CD pipeline using GitHub Actions to securely build, test, package (Docker), and deploy a Node.js REST API to Kubernetes.

  • Developed a secure backend microservice in Express.js, fortified with Helmet (HTTP headers), CORS, Morgan (logging), and automated Jest unit tests.
  • Engineered a "Shift-Left" CI/CD workflow that automatically triggers on every code push to compile the application and block failing tests.
  • Integrated aggressive security gating using Aqua Trivy (SAST/Container scanning) and npm audit (SCA) to physically block critical vulnerabilities and outdated dependencies from reaching production.
  • Automated Kubernetes deployments by actively validating infrastructure manifests (deployment.yaml) during the pipeline run to ensure zero-downtime releases.
