☁️ Cloud & DevOps Projects

1. Portfolio Website Infrastructure Deployment (Real DevSecOps Flow)

Developed and deployed this very multi-page professional portfolio website on AWS using Infrastructure as Code (IaC) and an automated CI/CD lifecycle.

  • AWS + Terraform (Production-Ready Architecture): Wrote real Terraform code to provision a custom AWS network from scratch, including a custom VPC (10.0.0.0/16), public subnets, an Internet Gateway, route tables, and strict Security Groups (HTTP + SSH).
  • EC2 Automation: Configured a Terraform user_data bash script to automatically install Node.js, configure `iptables` port forwarding (80 -> 3000), clone the repository, and start the app process using PM2.
  • CI/CD + AWS (Real DevOps Workflow): Built a fully automated CI/CD pipeline using GitHub Actions. Upon every code push, the pipeline authenticates via SSH, pulls the latest code to the EC2 instance, and automatically restarts the production server to achieve zero-downtime continuous deployment.

2. DevSecOps Automated Deployment Pipeline

Built a fully automated DevSecOps CI/CD pipeline using GitHub Actions to securely build, test, package (Docker), and deploy a Node.js REST API to Kubernetes.

  • Developed a secure backend microservice in Express.js, fortified with Helmet (HTTP headers), CORS, Morgan (logging), and automated Jest unit tests.
  • Engineered a "Shift-Left" CI/CD workflow that automatically triggers on every code push to compile the application and block failing tests.
  • Integrated aggressive security gating using Aqua Trivy (SAST/Container scanning) and npm audit (SCA) to physically block critical vulnerabilities and outdated dependencies from reaching production.
  • Automated Kubernetes deployments by actively validating infrastructure manifests (deployment.yaml) during the pipeline run to ensure zero-downtime releases.

3. AWS Static Website Hosting

Created an EC2 instance via the AWS Console to host a static website returning 'hello from aws'. Configured Security Groups to allow SSH, HTTP, and HTTPS traffic. Implemented comprehensive instance monitoring and uptime tracking using AWS CloudWatch.
